Hongyu(Chester) Yang

Hi, I am Chester, a solid Software Engineer with strong internship and projects experience. My Strengths include outstanding programming capability and systematic software skills in Front End, Back End, Databases, and Cloud Services.

In terms of my previous major projects and internship, I built user interfaces with basic HTML5, CSS, JavaScript. I also used popular frameworks like Angular and ReactJS. For server part, I applied Python Flask and Apache Tomcat + Java Servlet as web servers which were deployed in EC2 instances or Elastic Beanstalk. For databases, I played with relational database: MySQL, NoSQL databases: DynamoDB/BigTable, data search service Elastic Search. In order to do easy data analysis, Google Dataflow was used to dump data from BigTable to BigQuery. In order to get good product recommendation for users, I implemented a MapReduce algorithm to calculate the similarity between users in MapReduce at MongoDB. I wrote unit test cases for my code and used postman and Jmeter to test. I also analyzed running log with the help of Logstash and CloudWatch. I wrote RESTful APIs on swagger.io or directly designed on API Gateway to standardize the interface of project. Some of task functions were implemented in serverless service Lambda function and can easily be applied by any API Gateway deployment.
